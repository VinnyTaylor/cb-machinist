- ISO metric fine threads (M6-M20)

- Pitch displayed in mm for metric threads

**Input Validation**

**Comprehensive Tooling Recommendations**
- New Tooling tab in Reference section with sub-tabs for:

  - End Mills: Material-specific recommendations (flutes, helix, geometry)
  - Drills: Point angles, types, and coating by material
  - Inserts: Lathe insert grades, geometry, chipbreaker recommendations
- Integrated tooling suggestions in Speeds & Feeds calculator
  - Shows recommended end mill specs in mill mode
  - Shows recommended insert specs in lathe mode
  - Updates dynamically based on selected material

**Input Validation (Extended)**

**Favorites/Presets System**
- Save and load favorite settings in Speeds/Feeds calculator
- Separate presets for mill and lathe modes

- Shows preview of saved settings (material, diameter, flutes)

### 2D G-Code Simulator (Haas-style)
- Canvas 2D visualization (lightweight, no Three.js)
- G-code parser supporting:
  - Motion: G0 (rapid), G1 (linear), G2/G3 (arcs)
  - Plane selection: G17/18/19
  - Units: G20/21 (defaults to inches)
  - Distance mode: G90/91
  - Arc interpolation with I, J offsets
- Mill mode: XY plane view
- Lathe mode: ZX plane view (Z horizontal, X vertical)
- Haas-style line-by-line execution:
  - Current segment highlighted in yellow
  - Past segments shown in green
  - Future segments dimmed
- Features:
  - RESET, SINGLE BLOCK, CYCLE START, FEED HOLD buttons
  - Speed slider for playback speed
  - DRO overlay (X, Y, Z position)
  - Status display (RUNNING/READY/STOPPED, block count)
  - Mill/Lathe mode toggle
  - Sample programs dropdown (4 programs)
  - Tool crosshair with glow effect
  - Auto-centering with aspect ratio preservation
- Lazy loaded (~11 KB chunk)
