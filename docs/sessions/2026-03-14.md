# Session: 2026-03-14

## Summary
Major feature additions: Drill Point calculator, reset buttons, metric thread support, input validation, and comprehensive tooling recommendations.

## Work Done

### New Features

**Drill Point Calculator** (7th calculator)
- Calculates drill point length based on diameter and point angle
- Shows total drill depth needed for desired flat-bottom depth
- Common point angle reference table (118°, 135°, 90°, etc.)
- Calculates point volume
- Input validation with error messages

**Reset Buttons**
- New `ResetButton` component for consistent UI
- Added to all 7 calculators
- Resets inputs to sensible defaults

**Metric Thread Support**
- ISO metric coarse threads (M3-M24)
- ISO metric fine threads (M6-M20)
- Thread system toggle: Imperial / Metric / Metric Fine
- ISO 60° thread profile option
- Pitch displayed in mm for metric threads

**Input Validation**
- Created validation utility (`src/utils/validation.ts`)
- Error messages for invalid inputs
- Validates ranges, negative values, impossible geometry
- Added to: Drill Point, Taper calculators

**Expanded Materials Database (19 → 31)**
- Aluminum: Separated 6061/7075, added 2024, 5052
- Tool steels: A2, D2, O1, H13, S7
- Superalloys: Hastelloy C-276, Waspaloy, Stellite 6
- Cast iron: Added ductile iron

**Comprehensive Tooling Recommendations**
- New Tooling tab in Reference section with sub-tabs for:
  - Coatings: 7 coating types (TiN, TiCN, TiAlN, AlTiN, DLC, ZrN, Uncoated)
  - End Mills: Material-specific recommendations (flutes, helix, geometry)
  - Drills: Point angles, types, and coating by material
  - Inserts: Lathe insert grades, geometry, chipbreaker recommendations
- Integrated tooling suggestions in Speeds & Feeds calculator
  - Shows recommended end mill specs in mill mode
  - Shows recommended insert specs in lathe mode
  - Updates dynamically based on selected material
